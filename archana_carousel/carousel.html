The aim of this task is to build a carousel similar to http://sorgalla.com/jcarousel/

The Carousel code should be written as a Class so it may be instantiated a number of times. More than one carousel should be able to co-exist on a page at one time (hint: avoid giving the left and right arrow an id).

This link will be worth investigating http://api.jquery.com/find/ for lots of reasons which will become apparent as you build the app.

JQuery may be used and a link has been added to the head to the Google hosted JQuery CDN (you may want to change it to a local version if developing without an internet connection).

You don't, at this stage, need to create any DOM elements programatically, but if you prefer to feel free. The HTML may have Divs, Classes and Id added to it as you see fit.

There are no right or wrong ways to approach the task but think ease of use, flexibility and re-usability when looking for a solution and avoid too mani unique IDs on divs.
